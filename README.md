# Cal-ITP Benefits

![Cal-ITP Benefits Mockup1200x900](https://github.com/user-attachments/assets/3f4e9e2f-0cae-4c4b-99b2-49c580da3c27)


<a href="https://benefits.calitp.org" target="_blank">Cal-ITP Benefits</a> helps riders of participating transit systems verify their eligibility for a transit benefit and link that status to their contactless debit or credit card so they automatically receive reduced fares when they tap to pay.

This web application is open-source software that is designed, developed, and maintained by <a href="https://compiler.la/" target="_blank">Compiler LLC</a> on behalf of the California Department of Technology, Caltrans, and our agency partners.

View technical documentation online at <https://docs.calitp.org/benefits>.
